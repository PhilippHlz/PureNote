# True Note OOSL 
Im Rahmen der EA3 im Modul OOSL möchten wir als Gruppe das Projekt TrueNote präsentieren.
Dabei handelt es sich um einen selbst entwickelten **Text-Editor mit Tkinter-Oberfläche.**

TrueNote enthält zusätzlich eine **To-Do-Liste**, die sich direkt auf bestimmte Textabschnitte bezieht, sowie einen **Timer** zur Unterstützung von Pausen und Fokusphasen.
Die To-Dos können einzelnen Absätzen oder Bereichen des Textes zugeordnet werden, sodass klar sichtbar ist, welche Teile noch überarbeitet werden müssen.<br>
Der integrierte Timer arbeitet in festen Intervallen von 30 Minuten Arbeit und 5 Minuten Pause. Nach Ablauf der 30-minütigen Fokuszeit startet automatisch die 5-minütige Pause, bevor der nächste Arbeitszyklus beginnt. Dadurch entsteht ein gleichmäßiger Arbeitsrhythmus, der Konzentration, Struktur und Erholung kombiniert.

Durch die Verbindung von Textbearbeitung, abschnittsbezogenen To-Dos und dem Intervall-Timer wird TrueNote zu einem kompakten Werkzeug für strukturiertes und effizientes Arbeiten.
Wir überlegen eventuell noch einen Musikspieler einzubauen und gegebenenfalls noch ein Exportmöglichkeit für ein HTML-Dokument, damit der Text auf jedem Endgerät geöffnet werden kann.
